# gh-pages-deploy-workflow
Simple GitHub Actions workflow to deploy a static website to GitHub Pages.

## Project URL 
https://roadmap.sh/projects/github-actions-deployment-workflow

## Project Details
This project is to learn the notion of continuous integration and continuous deployment. Write a simple GitHub Actions workflow to deploy a static website to GitHub Pages.

## Project Goals
- Learn the notion of continuous integration and continuous deployment.
- Write a simple GitHub Actions workflow to deploy a static website to GitHub Pages.

## Project Structure
- `README.md`: Project description and details.
- `.github/workflows/deploy.yml`: GitHub Actions workflow for deployment.
- `index.html`: Sample static website content.

## Project Setup
- Create a new repository.
- Enable GitHub Pages for the repository.
- Push the `index.html` file to the repository.
- Add the GitHub Actions workflow file to the `.github/workflows` directory.
